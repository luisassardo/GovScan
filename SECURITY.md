# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in this project, please report it privately.

**Contact:** journalists_opsec@proton.me  
**PGP:** Available on request.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I aim to respond within 72 hours and will keep you informed of the fix timeline.

## Out of Scope

- Vulnerabilities in third-party dependencies (report directly to them)
- Social engineering attacks
